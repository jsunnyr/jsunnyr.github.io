Linux硬件管理

总核数 = 物理CPU个数 X 每颗物理CPU的核数
总逻辑CPU数 = 物理CPU个数 X 每颗物理CPU的核数 X 超线程数

一般情况，我们认为一颗cpu可以有多核，加上intel的超线程技术(HT)。

Linux cat /proc/cpuinfo 具有相同core id的CPU是同一个core的超线程，具有相同physical id的CPU是同一个CPU封装的线程或核心。
